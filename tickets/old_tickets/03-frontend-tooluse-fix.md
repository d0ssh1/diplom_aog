Я вижу по скриншотам, в чем проблема. Claude сделал для вас классический «современный темный интерфейс» (Modern Dark Mode): скругленные углы, мягкие тени, полупрозрачные серые фоны-плашки и стандартные отступы. 

Но наш дизайн — это **Technical UI / Brutalism (Технический брутализм)**. Он опирается на эстетику профессионального инженерного софта, чертежей и интерфейсов командной строки.

Чтобы Claude (или другой ИИ) выдавал нужный результат, ему нужно задать четкие правила стилизации (мы используем Tailwind CSS). Вот **4 главных правила** и примеры кода, которые нужно передать вашему ИИ:

### Правило 1: Строгий запрет на скругления (Zero Border Radius)
В брутализме нет мягкости. Все элементы должны быть жесткими прямоугольниками.
* **Ошибка на скрине:** У кнопок есть `rounded-lg` или `rounded-md`.
* **Как надо:** Полное отсутствие скруглений. Если используете UI-библиотеки, принудительно ставьте `rounded-none`.

### Правило 2: Жесткие рамки вместо мягких фонов (Borders over Backgrounds)
Вместо того чтобы заливать неактивные кнопки серым цветом, мы оставляем их прозрачными (цвет фона панели), но обводим тонкими рамками (`border-zinc-800`).
* **Ошибка на скрине:** Неактивные кнопки залиты серым цветом (выглядят как громоздкие блоки).
* **Как надо:** Кнопка обведена рамкой. Иконка внутри кнопки лежит в **своем собственном жестком квадрате**.

### Правило 3: Моноширинные шрифты и "код"-акценты (Monospace & Tech Accents)
Это то, что дает вайб "хакерского/инженерного" софта.
* **Заголовки секций:** Должны быть написаны капсом, с широким межбуквенным расстоянием и начинаться с `//`.
* **Шрифт:** Обязательно добавлять класс `font-mono`.

### Правило 4: Высококонтрастный Hover-эффект
Когда мы наводим на элемент, он не должен просто стать "чуть светлее". Он должен резко вспыхнуть сигнальным оранжевым (`#FF4500`). Иконка должна поменять цвет (например, с оранжевого на черный для контраста).

### Примеры кода (Tailwind CSS), которые нужно скормить Claude:

Покажите этот код вашему ИИ и скажите: *"Сделай панель инструментов в точности по этому паттерну стилей Tailwind"*.

#### 1. Заголовок секции (например, "// РЕДАКТОР СТЕН")
```jsx
{/* Строгий моноширинный заголовок с префиксом-комментарием */}
<h4 className="font-mono text-xs text-zinc-500 mb-4 uppercase tracking-widest">
  {'// Редактор стен'}
</h4>
```

#### 2. НЕАКТИВНАЯ кнопка инструмента (Нарисовать стену)
Обратите внимание на структуру: сама кнопка прозрачная с серой рамкой, а иконка лежит в черном квадрате.
```jsx
<button className="w-full flex items-center p-3 border border-zinc-800 text-zinc-400 hover:border-[#FF4500] hover:text-[#FF4500] transition-all group bg-transparent rounded-none outline-none">
    
    {/* Квадрат с иконкой внутри кнопки */}
    <div className="w-10 h-10 flex items-center justify-center mr-4 bg-black group-hover:bg-[#FF4500] group-hover:text-black transition-colors">
        <PenLine size={20} />
    </div>
    
    {/* Текст кнопки */}
    <span className="font-mono text-sm tracking-wide">Нарисовать стену</span>
</button>
```

#### 3. АКТИВНАЯ кнопка инструмента (Кадрирование)
Если инструмент выбран, рамка становится оранжевой, появляется легкая оранжевая заливка, а квадрат с иконкой становится полностью сигнально-оранжевым с черной иконкой.
```jsx
<button className="w-full flex items-center p-3 border border-[#FF4500] bg-[#FF4500]/10 text-[#FF4500] transition-all group rounded-none outline-none">
    
    {/* Инвертированный квадрат с иконкой */}
    <div className="w-10 h-10 flex items-center justify-center mr-4 bg-[#FF4500] text-black">
        <Crop size={20} />
    </div>
    
    {/* Текст (жирный) */}
    <span className="font-mono text-sm font-bold tracking-wide">Кадрирование</span>
</button>
```

#### 4. Ползунок (Range Slider)
На вашем скриншоте у Claude ползунок стандартный, белый и круглый. Наш ползунок должен быть квадратным.
```jsx
<div className="flex items-center space-x-4 px-2">
    <input 
        type="range" 
        min="1" 
        max="30" 
        defaultValue="6"
        className="w-full h-1 bg-zinc-800 appearance-none outline-none cursor-pointer z-20 
                   [&::-webkit-slider-thumb]:appearance-none 
                   [&::-webkit-slider-thumb]:w-4 
                   [&::-webkit-slider-thumb]:h-6 
                   [&::-webkit-slider-thumb]:bg-white 
                   [&::-webkit-slider-thumb]:rounded-none" 
        style={{ 
            background: `linear-gradient(to right, #FF4500 20%, #27272a 20%)` 
        }} 
    />
    <span className="font-mono text-xs w-10 text-right text-zinc-400">6 px</span>
</div>
```
